# My-Giphy

## - It fetch's gifs from [Giphy](https://developers.giphy.com/docs/api/)

## - Libraries/Packages I used
   - [axios](https://www.npmjs.com/package/react-native-axios)
   - [react-navigation](https://reactnavigation.org/docs/getting-started/)
   - [fast-image](https://www.npmjs.com/package/react-native-fast-image)
   - [toast](https://www.npmjs.com/package/react-native-simple-toast)

## - Run *npm install* to install all the packages or install packages separately
   - If you haven't setup react native environment then [click here](https://reactnative.dev/docs/environment-setup)
